## 2. Description of the Plugin

HASS.agent-mic-state-plugin is a free, open source tool that reads Discord voice channel state via Discord's local RPC API and exposes it as a binary sensor to a locally-hosted Home Assistant instance. It is intended solely for personal, non-commercial home automation use.
